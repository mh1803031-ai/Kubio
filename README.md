## 🚀 Demo

## ✨ Fitur Utama
- **AI-Driven Design:** Pembuatan layout otomatis berdasarkan input pengguna.
- **Fully Responsive:** Tampilan optimal di perangkat mobile, tablet, maupun desktop.
- **Fast Performance:** Dioptimalkan untuk kecepatan akses tinggi menggunakan Netlify Edge.
- **Customizable Blocks:** Komponen UI yang mudah dimodifikasi sesuai kebutuhan.
